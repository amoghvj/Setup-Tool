This directory simulates the installed application layout (mirrors Program Files\SetupTool).

After building the exe, copy the following into this directory:
  - dist/SetupTool.exe  →  installed/SetupTool/SetupTool.exe
  - resources/server/    →  installed/SetupTool/resources/server/

Then run SetupTool.exe from here to test as-if-installed behaviour.
